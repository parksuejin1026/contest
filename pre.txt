1. 본인이 만들고자 하는 앱 or 웹 한 줄 설명

AI가 종이 계약서를 실시간으로 스캔 및 분석하여 위험 조항과 불리한 계약 조건을 탐지하고 쉽게 설명해주는 AI 계약서 분석 키오스크 시스템

2. 프로젝트 기획 이유

계약서는 법률 용어와 복잡한 문장 구조로 이루어져 있어 일반 사람들이 내용을 정확하게 이해하기 어렵다. 특히 대학생이나 사회초년생들은 아르바이트 계약서, 외주 계약서, 임대차 계약서 등을 충분히 이해하지 못한 상태에서 서명하는 경우가 많다.

또한 기업에서도 계약서 검토에는 많은 시간이 소요되며, 중요한 위험 조항을 사람이 직접 확인해야 하는 문제가 존재한다.

최근 생성형 AI와 OCR 기술이 발전하면서 종이 문서를 자동으로 읽고 분석하는 것이 가능해졌기 때문에, 이를 활용하여 계약서를 빠르게 분석하고 위험 요소를 직관적으로 확인할 수 있는 AI 계약서 분석 시스템을 기획하고자 하였다.

3. 프로젝트 목표
3-1) 종이 계약서 스캔 시스템 구현
3-2) OCR 기반 계약서 텍스트 추출 기능 구현
3-3) AI 기반 위험 조항 탐지 기능 구현
3-4) 계약 핵심 내용 자동 요약 기능 구현
3-5) 쉬운 말 설명 기능 구현
3-6) 위험도 시각화 및 LED 경고 시스템 구현

최종적으로는 사용자가 종이 계약서를 키오스크에 올려두면 AI가 자동으로 계약서를 분석하고 위험 요소를 직관적으로 보여주는 시스템을 
만드는 것이 최종 목표이다.

4. 프로젝트 주요 기능
4-1) 계약서 스캔 기능 : 웹캠을 이용하여 종이 계약서를 촬영하는 기능
4-2) OCR 텍스트 추출 기능 : 촬영된 계약서 이미지에서 글자를 추출하는 기능
4-3) 위험조항 탐지 기능 : AI가 계약서를 분석하여 다음과 같은 위험 요소 탐지
- 자동 연장 조항
- 과도한 위약금
- 계약 해지 제한
- 책임 범위 과다
4-4) 계약 핵심 요약 기능 : 긴 계약서를 AI가 핵심 내용 중심으로 자동 요약
ex) 계약 기간 1년, 중도 해지 시 위약금 존재, 자동 연장 가능성 있음
4-5) 쉬운 말 설명 기능 : 복잡한 법률 문장을 일반인이 이해하기 쉬운 표현으로 변환
ex) “본 계약과 관련한 분쟁 발생 시 관할 법원은 갑의 소재지를 관할하는 법원으로 한다.” -> “문제가 생기면 상대방 회사 지역의 법원에서 재판해야 할 수 있습니다.”
4-6) 위험도 LED 표시 기능 : 계약 위험도에 따라 RGB LED 색상을 변경
- 안전 : 초록 LED
- 주의 : 노랑 LED
- 위험 : 빨강 LED

5. 구체적인 구현 방법
이 프로젝트는 AI 계약서 분석 키오스크 형태로 구현한다.

사용자는 종이 계약서를 키오스크 문서 거치대에 올려둔다. 이후 웹캠이 계약서를 촬영하고 OpenCV를 이용하여 이미지 보정 작업을 수행한다.

보정된 이미지는 OCR(Tesseract OCR)을 사용하여 텍스트를 추출한다.

추출된 계약 내용은 OpenAI API 또는 로컬 LLM 모델을 이용하여 분석한다. AI는 계약 문장 중 다음과 같은 위험 문구를 탐지한다.

- 자동 갱신
- 위약금
- 계약 해지 제한
- 과도한 책임 조항

이후 AI는 다음 기능을 수행한다.

- 위험 조항 분석
- 핵심 계약 내용 요약
- 쉬운 말 설명
- 위험도 계산

분석된 결과는 React 기반 웹 화면에 표시되며, 위험 문장은 빨간색으로 하이라이트된다.

또한 계약 위험도에 따라 Arduino 기반 RGB LED 색상이 변경된다.

- 안전 : 초록 LED
- 주의 : 노랑 LED
- 위험 : 빨강 LED

시스템 전체 흐름

종이 계약서 업로드 → 웹캠 촬영 → OpenCV 이미지 보정 → Tesseract OCR 텍스트 추출 → LLM 기반 계약 분석 및 위험도 계산 → SQLite 데이터베이스 저장 → 아두이노 시리얼 통신을 통한 RGB LED 제어 → React 화면 결과 시각화 → 열전사 프린터를 이용한 분석 리포트 출력

6. 사용 기술

Frontend : React, Vite
→ 계약서 업로드 화면, 분석 결과 화면, 위험도 그래프, 출력 버튼 UI 구현

Backend : FastAPI, Python
→ OCR 처리, AI 분석 요청, 위험도 계산, Arduino 및 프린터 제어 담당

AI 분석 : OpenAI API 또는 Ollama(Local LLM)
→ 위험 조항 탐지, 계약 요약, 쉬운 말 설명 생성 수행

OCR : Tesseract OCR
→ 계약서 이미지에서 텍스트 추출

이미지 처리 : OpenCV
→ 촬영된 계약서 이미지 보정 및 OCR 인식률 향상

PDF 및 문서 처리 : PyPDF
→ PDF 계약서 내부 텍스트 추출

데이터베이스 : SQLite
→ 계약 분석 결과 및 위험도 데이터 저장

시각화 : Chart.js
→ 계약 위험도를 그래프 및 게이지 형태로 시각화

7. 기대 효과

7-1) 계약서 이해도 향상 : 일반 사용자도 어려운 계약 내용을 쉽게 이해 가능
7-2) 계약 검토 시간 감소 : AI가 위험 조항을 자동 탐지하여 검토 시간 단축 가능
7-3) 사회초년생 피해 예방 : 불리한 계약 조항을 빠르게 확인 가능
7-4) AI 기반 문서 자동화 가능성 : 향후 기업 계약 검토 시스템으로 확장 가능
7-5) 직관적인 위험 인식 가능 : LED 및 시각화를 통해 계약 위험도를 쉽게 파악 가능

8. 프로젝트 차별점

기존에도 AI 기반 계약 분석 및 문서 요약 서비스는 존재한다. 대표적으로 DocuSign Insight, Luminance, Lawgeex와 같은 서비스들이 있으며, 최근에는 ChatGPT나 Notion AI 같은 생성형 AI를 활용하여 문서를 요약하거나 분석하는 사례가 많아지고 있다.

하지만 기존 서비스들은 대부분 기업 및 법률 전문가 중심으로 설계되어 있어 일반 사용자가 사용하기 어렵고, 복잡한 법률 용어를 그대로 제공하는 경우가 많다. 

또한 대부분 디지털 문서 기반 서비스이며, 실제 종이 계약서를 즉시 분석하고 직관적으로 위험도를 보여주는 시스템은 부족하다.

본 프로젝트는 단순 문서 요약이 아닌 “계약 위험 분석”에 초점을 맞춘다는 점에서 차별성을 가진다.

특히 다음과 같은 차별 요소를 가진다.

8-1) 위험 조항 탐지 기능 : AI가 계약서 내 위험 가능성이 높은 문장을 자동 탐지한다.
ex)
- 자동 연장 조항
- 과도한 위약금
- 계약 해지 제한
- 책임 범위 과다
8-2) 쉬운 말 설명 기능 : 복잡한 법률 문장을 일반 사용자도 이해할 수 있는 쉬운 표현으로 변환한다.
8-3) 실시간 계약서 스캔 키오스크 : 종이 계약서를 웹캠으로 촬영하여 즉시 분석 가능
8-4) 하드웨어 기반 위험도 표시 : Arduino RGB LED를 활용하여 위험도를 직관적으로 표현
8-5) 일반 사용자 대상 서비스 : 대학생 및 사회초년생도 쉽게 사용할 수 있도록 설계

따라서 본 프로젝트는 단순 계약 요약 시스템이 아니라, 일반 사용자가 계약 내용을 쉽게 이해하고 위험 요소를 직관적으로 확인할 수 있도록 도와주는 AI 기반 계약 분석 키오스크 시스템이라는 점에서 차별성을 가진다.

9. 실제 제작하였을 때 시연 방식

1단계 : 키오스크 실행
사용자가 계약서 분석 키오스크 앞에 위치한다.

2단계 : 종이 계약서 업로드
사용자가 계약서를 문서 거치대 위에 올려둔다.

3단계 : 계약서 촬영
웹캠이 계약서를 촬영한다.

4단계 : OCR 및 AI 분석
시스템이 자동으로 다음 작업을 수행한다.

-이미지 보정
-OCR 텍스트 추출
-AI 위험 조항 분석

5단계 : 위험 조항 탐지
AI가 자동 연장 조항, 위약금 조항, 책임 범위 과다 등을 탐지하고 위험 문장을 빨간색으로 표시한다.

6단계 : 계약 핵심 요약 출력

-계약 기간 : 1년
-자동 연장 가능
-중도 해지 시 위약금 존재

7단계 : 쉬운 설명 기능
사용자가 위험 문장을 클릭하면 쉬운 설명을 제공한다.

예시)
"본 계약과 관련한 분쟁 발생 시 관할 법원은 갑의 소재지를 관할하는 법원으로 한다."
→ "문제가 생기면 상대방 회사 지역의 법원에서 재판해야 할 수 있습니다."

8단계 : 위험도 LED 출력
계약 위험도에 따라 RGB LED 색상이 변경된다.

-안전 : 초록 LED
-주의 : 노랑 LED
-위험 : 빨강 LED

9단계 : 최종 결과 화면 출력
위험 조항, 계약 요약, 쉬운 설명, 위험도 그래프를 화면에 시각적으로 출력한다.

10단계 : 결과 출력 버튼 클릭
사용자가 결과 출력 버튼을 누르면 계약 분석 리포트가 열전사 프린터를 통해 종이 형태로 출력된다.
